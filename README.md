## вҡҷпёҸ Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/your-username/cbse-admission-system.git
```

2. **Move project to server**

* Copy project folder to:

  * `htdocs` (XAMPP) OR
  * `www` (WAMP)

3. **Start Server**

* Start Apache & MySQL from XAMPP/WAMP

4. **Database Setup**

* Create a database in **phpMyAdmin**
* Update credentials in:

```

```

5. **Run Project**

```
http://localhost/CBSE School Admission System/html/home.html
```
